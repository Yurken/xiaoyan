# 获取帮助

- 使用方法和一般问题：前往 [Discussions](https://github.com/Yurken/xiaoyan/discussions)；
- 可复现缺陷：使用 [Bug Report](https://github.com/Yurken/xiaoyan/issues/new?template=bug_report.md)，并附版本、系统、复现步骤和已脱敏日志；
- 功能建议：使用 [Feature Request](https://github.com/Yurken/xiaoyan/issues/new?template=feature_request.md)，说明真实场景与预期结果；
- 安全漏洞：不要公开提交，按照 [安全政策](SECURITY.md) 使用私有 Security Advisory；
- 模型服务计费、配额、内容政策与可用性：联系对应第三方服务商。

公开求助前请移除 API Key、Cookie、绝对路径、未公开论文内容、真实研究数据、数据库、配置导出文件和个人信息。
