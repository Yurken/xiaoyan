# 社区行为准则

我们致力于提供开放、友善、无骚扰的协作环境，不因经验、身份、背景、能力或观点不同而排斥参与者。

## 期望行为

- 尊重不同意见，聚焦事实、代码和具体问题；
- 给出具体、可执行的反馈，并善意理解他人的表达；
- 承认错误并及时修正；
- 保护他人的隐私、凭据与未公开研究成果；
- 在公开发布日志、截图或数据前完成脱敏。

## 不可接受的行为

- 人身攻击、骚扰、歧视、威胁或持续扰乱讨论；
- 未经许可公开他人的身份信息、凭据或研究材料；
- 发布垃圾内容、恶意代码或故意误导他人；
- 其他不适合专业协作环境的行为。

## 执行

维护者可以编辑或移除不当内容，并根据严重程度采取提醒、警告、临时限制或永久禁止参与等措施。

如需私下报告社区行为问题，请使用维护者 GitHub 个人资料中公布的私密联系方式。若暂时没有可用的私密渠道，可以创建一个不包含事件细节的 Issue，请求维护者提供私下联系方法。所有报告会在必要范围内保密处理。
