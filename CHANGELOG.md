# 更新日志

## [0.3.0] - 2026-04-12

### 新增
- 知识图谱与 Graph RAG 功能：基于论文关联关系构建图结构，支持图增强检索（Graph RAG）
- Copilot Agent 状态图重构：Agent 工作流改为显式状态机，执行流程更清晰可控
- PPT 幻灯片预览面板：PPT 生成后可在工作区内直接预览各页幻灯片，完善生成工作流

### 优化
- 知识图谱工作区：重构布局，新增下拉切换组件，研究兴趣与助手设置拆分为独立模块
- 拟态主题基础组件统一：Button、Input、Select 等基础组件全面收敛到拟态设计规范

### 修复
- 修复 `packages/ui/Select` 在 Next.js App Router 中缺少 `"use client"` 声明导致 Web 端构建失败

---

## [0.2.8] - 2026-03-23

### 新增
- 论文批量导入：选文件对话框支持多选 PDF，串行处理并显示实时进度（"导入中 2/5"），单文件失败不中断整体
- 实用工具页签化：arXiv 检索、刊会查询、科研友链改为页签切换
- 小妍解读公式渲染：提示词要求 LaTeX 格式，KaTeX 正常渲染行内与块级公式
- 复现指南增强：无开源代码时主动给出替代复现方案、数据集下载链接及环境配置步骤

### 优化
- 小妍解读布局：各分析维度改为彩色左边框卡片 + 胶囊标签，内容区使用 Markdown 完整渲染
- 导航"对话"改为"小妍"，相关页面文案统一
- 时间戳精确到秒
- arXiv 排序方式选择器改为 neumorphic 按钮组，与整体风格一致
- 笔记标签仅展示 AI 生成标签并标注"小妍"来源徽章，移除手动输入入口
- macOS Dock 图标与窗口图标保持一致

### 修复
- arXiv 500 错误：`submittedDate` 范围查询中 `+TO+` 被 URL 编码为 `%2B` 导致 Lucene 解析失败
- Markdown 渲染：分析内容区 `text-xs` 覆盖 prose 排版，颜色系统改用 `ink-*` 设计 token
