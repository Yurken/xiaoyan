# 更新日志

以下内容面向日常使用，主要记录你能直接感知到的变化，不展示内部实现细节。

## [0.4.6.1] - 2026-07-10

### 新增
- Copilot 输入区支持生成中终止，发送快捷键统一为 ⌘/Ctrl + Enter。
- 技能选择器拆分为独立组件，便于后续扩展与维护。
- 版本管理支持四段修订号（如 v0.4.6.1），自动映射为 Cargo/Tauri 兼容的 SemVer 预发布版本。

### 优化
- 研究主题画像字段兼容旧数据库中偶尔以单个字符串保存的数据。

### 修复
- 忽略本地研究导入与展示生成的临时文件，避免误提交。

## [0.4.6] - 2026-07-09

### 新增
- 论文库排序交互重构：拖拽排序改为按钮排序，支持按名称、重要性正逆序切换。
- 论文分析全 Agent 引入图表上下文，解读与复现分析时能结合论文中的图表信息。
- 新增论文图表提取能力，为后续图表解读与复现验证提供基础。
- 翻译面板支持模型选择，可按需切换翻译模型。

### 优化
- PDF 阅读页划词选区不再抖动，缩放操作更流畅。
- 更新通知与自动更新提示更清晰。
- 论文弹窗按钮统一为拟态风格，复现/验证指南与论文笔记按钮统一为图标绿色，视觉更一致。

### 修复
- 修复 KDD'20 等带年份后缀的 venue 无法匹配 CCF A 的问题。

## [0.4.5] - 2026-07-02

### 新增
- 集成 nature-reader 与 nature-polishing 技能，论文解读与笔记生成更贴合学术写作场景。
- 论文卡片新增笔记入口，可选择让小妍基于论文内容生成结构化笔记，也可手动整理；已创建的笔记可直接查看或跳转到知识库。
- 知识笔记支持导入本地 Markdown/文本文件，新建、编辑与阅读改为全屏页面，知识整理空间更宽敞。
- 论文阅读页支持拖拽调整面板宽度，切换论文时会保存并恢复缩放比例与滚动位置，连续阅读更连贯。
- 论文解读完成后，解读按钮显示为绿色「已解读」状态，当前进度一目了然。
- 设置页新增「启用主动研究」开关，可控制小妍是否自动扫描并推荐研究线索。
### 优化
- 聚焦模式各页面观感继续优化，信息层级更清爽。
- 对话列表去除拟态样式并缩小间距，长时间浏览更轻松。

### 修复
- 修复小妍协作流程与综述交互问题，复杂任务执行与报告展示更稳定。
- 持久化综述与规划运行状态，避免切换页面或中断后进度丢失。
- 修复桌面端数据切换竞态，减少会话与投稿页面的状态错乱。
- 修复综述参考文献列表不换行的问题。
- 修复「启用主动研究」开关无法持久化的问题。
- 修复保存权限相关异常，避免部分写入失败。
- 修复工作台提示逻辑，优先引导继续精读未分析论文。
- 修复知识笔记数据兼容性问题。
- 改善阅读器选中文字工具栏体验，减小解读面板段落和分隔线间距。

## [0.4.4] - 2026-06-28

### 新增
- 小妍对话接入联网搜索：基于 Tavily 提供实时网络检索，支持多 Key 轮换、独立开关与连接测试；搜索改为 function calling 工具方式触发，结果更精准。论文检索也会补充网络结果，单次返回篇数上限放宽至 50。
- 主动研究员的推荐论文支持一键下载导入：在小妍伴侣的推荐抽屉里可直接把 arXiv 论文下载并导入论文库，导入后论文标题与文件名都以论文标题命名。
- 对话中输入斜杠即可唤起技能并填写变量，常用操作更顺手。
- 长期记忆升级为语义检索，相关记忆更容易被准确召回，同时补齐存储健康治理。
- 设置页模型配置支持品牌图标、向量模型与可查询模型下拉，配置各角色模型更直观。
- 小妍设置新增 token 用量与字符统计，方便随时掌握消耗。
- 升级页与设置页新增意见反馈入口，反馈会自动附带系统版本信息，便于跟进。
- 论文卡片右键菜单新增「在访达/文件管理器中打开」，定位本地 PDF 更方便。
- 首次打开的快速开始引导改为弹窗形式，并去掉第三步的任务感，初次上手更轻松。

### 优化
- 设置页 WebDAV 同步相关卡片合并收口，并补回测试连接入口，同步配置更集中。
- 步骤协作配置面收敛命名、统一组织，查找与切换配置更省事。
- 阅读笔记面板拆分为独立组件，笔记的展示与后续维护更清晰。

### 修复
- 修复知识图谱普通滚轮误触缩放、连线端点位置偏移的问题。
- 恢复论文重要性标记为顶部彩色横线，替代原先的小圆点显示。
- 修复投稿诊断与证据链路对齐数据库与前端契约的若干问题。
- 修复小妍对话发送、会话切换、思考面板的一批稳定性问题，并解决后端与联网搜索耦合导致的异常。
- 修复实验记录会话项嵌套按钮产生的水合告警。
- 修复 Web 端默认设置缺失联网搜索字段导致的问题。
- 同步发版流水线类型检查与 web 构建，并补齐 UI 依赖声明，提升发布可靠性。

## [0.4.3] - 2026-06-25

### 新增
- 桌面端、Web 与移动端小妍对话正式支持图片：可粘贴、拖拽或直接发送图片，对话气泡内可直接查看，同会话多轮上下文也会保留图片。
- 对话图片默认走独立的「视界」视觉模型；若未配置视觉模型，会给出明确提示，并支持在设置页一键测试视觉接口连通性。
- 移动端新增「给小妍资料找 idea」能力，把现有论文、笔记等材料交给小妍，辅助发掘可继续深入研究的方向。
- 论文库新增「改写」功能，选中语料后可快速生成不同表述；支持合并重复论文，并允许文件夹建立子文件夹与跨层归档。
- 阅读批注升级：支持绘制形状、填充与移动，划词翻译改为单条完整面板，PDF 首屏前几页发虚的问题得到修复。
- 研究主题规划接入各模块真实进展，进展阶梯上方移除冗余进度条，让当前推进状态更直观。
- 设置页角色模型支持独立「测试连接」，可针对每个角色单独验证模型、接口地址和密钥是否可用。
- 发布流水线新增 Linux 构建与分发；LaTeX 编译支持跨平台，并会按平台引导安装对应引擎。
- 桌面端统一抽取平台检测逻辑，macOS、Windows、Linux 的识别与后续处理更一致。
- PPT 导出补齐演讲者备注、页脚页码，并强化整体叙事结构。

### 优化
- 设置页重构：移除「设置」大标题，胶囊导航上移至顶栏；切换与管理入口移入弹窗，测试与保存整合到小妍角色卡并改为自动保存。
- 移除冗余的「默认回退」角色卡，角色配置更聚焦；界面风格锁定为拟态并隐藏风格切换，视觉表达更统一。
- Markdown 整理工具改用分隔行输出并增强代码块感知分块，长内容整理结果更干净。
- MorphingText 按行首禁则分组，避免标点落在行首；对话回答的 Markdown 排版进一步优化。
- 阅读工具栏按钮统一为拟态风格，并互换「返回」与「论文库开关」位置，与整体交互习惯一致。
- 桌面端拖拽体验更稳定：FocusHome 改用幂等清理，拖拽监听细分事件并暴露位置。
- 自定义接口地址容错增强，自动处理常见误填的重复路径后缀。

### 修复
- 修复「视界」视觉模型在配置 Kimi Coding 等兼容接口时测试失败的问题。
- 修复视觉模型测试图因尺寸过小而部分接口拒绝解码的问题，测试图改用正常尺寸，兼容性更好。
- 修复对话中未发送的附件跨路由丢失、图片进入同步快照、多轮历史图片过量回放等问题。
- 修复 copilot 重试与编辑重发、流式搜索状态闭包等对话稳定性问题。
- 修复投稿管理中的模拟审稿导入回填、审稿流式监听重复注册、cover letter 与润色请求串写等问题。
- 修复设置导入、应用历史、恢复备份后触发多余自动回写的问题。
- 修复阅读器卸载时未中止划词解读流、桌面伴侣监听注册卸载竞态导致监听泄漏等问题。
- 修复移动端 WebDAV 同步在部分服务器响应格式下解析失败的问题。
- 修复 desktop type-check 闸门空跑并清零累积的类型错误；同时修复若干 lint 与格式化问题。
- 角色模型测试失败时，现在会在浏览器控制台输出详细错误，方便快速定位接口配置问题。

## [0.4.2] - 2026-06-17

### 新增
- 新增「自动同步」：填一次 WebDAV 账号即可在多台设备间自动同步，启动、切回前台和定时都会在后台静默进行，无需手动点上传/下载。
- 多设备同时使用时不再互相覆盖：各设备的新增与修改会按时间自动合并，删除也会同步到其它设备，不会出现旧数据「复活」。
- 同步密码仅保存在本机系统钥匙串，并用于端到端加密；附件（如 PDF）按内容去重，不会重复上传。
- 小妍协同流程进一步收束，当前研究任务中的步骤、状态和关键信息展示得更完整，连续推进复杂任务时更容易跟住过程。
- 设置与阅读相关体验继续补齐，常用入口和说明组织得更清楚，查找配置与回看阅读笔记更方便。
- 应用内更新日志继续完善，版本说明更适合直接给用户查看。

### 优化
- 桌面端协同台、设置页和首页的若干结构继续整理，减少不必要的信息噪音，让高频操作更聚焦。
- 小妍的对话与过程展示文案进一步统一，界面中对步骤、能力和状态的表达更自然。
- 阅读器中的笔记保存与读取逻辑在不同运行环境下更稳定，桌面端与非桌面端的行为切换更顺滑。

### 修复
- 修复若干桌面端任务流程中的状态不同步、信息衔接不顺和结果展示不稳定问题。
- 修复阅读器笔记在部分环境下可能回退到错误存储方式的问题，减少笔记读取不一致的情况。
- 改善桌面端发布前后的版本信息一致性与若干界面细节问题。
