# 60 秒旁白稿

从论文堆里，拉出清晰研究路线。

研究工作不只是问答。找方向、筛论文、读图表、写综述、沉淀知识、追踪投稿，每一步都可能断开。

小妍把这些环节接成一条可观测的 AI 协同流程。Supervisor 拆解任务，专长 Agent 分工执行，计划、轨迹与来源实时呈现。

上传 PDF，自动提取全文与图表，建立语义分块。精读、复现建议和带引用回答，不再散落在多个工具之间。

论文引用、知识卡片和用户记忆会进入知识图谱。Graph RAG 让小妍在回答复杂问题时看见上下文，也看见关联。

投稿阶段，DDL 日历、状态看板、版本控制、模拟审稿和作者回复统一管理。

所有核心数据保存在本地 Tauri 与 SQLite 内核中。你还可以按用途配置快速模型、深度分析模型、写作模型和视觉模型。
