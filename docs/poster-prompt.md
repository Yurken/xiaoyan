## 小妍 — 宣传海报提示词

---

### 完整提示词

```
一张简洁现代的桌面端 AI 研究助手「小妍」宣传海报。

构图：16:9 横版，左侧文字排版，右侧视觉元素。

背景：柔和的冷调蓝灰色垂直渐变，从顶部 #F0F4F8 过渡到底部 #E8ECF0。表面干净无纹理，呈现出类似 Apple 产品页面的高级感。

视觉风格：拟态（Neumorphic / Soft UI）设计语言。所有元素通过双向柔和投影营造蓬松的浮雕质感——右下方投射冷灰色（#C8CDD3）阴影，左上方叠加纯白（#FFFFFF）高光。画面中不存在任何硬边框，深度感完全由柔影对呈现。卡片圆角 24-28px，小元素 16px，按钮为胶囊形。

中心视觉：一位风格化的动漫风女性角色（吉祥物「小妍」）——短发棕色、大眼睛、穿白色连帽卫衣配蓝色抽绳、戴浅蓝色云朵形发饰。插画风格需与拟态画面融合——无硬描边，柔边缘，冷调着色，表情自信专注。

角色周围，六张拟态功能卡片以松散弧线排列漂浮，每张卡片具备：
- 145° 线性渐变背景，从近白色（#F8FAFC）到浅冷灰（#E6EAF0）
- 双向柔影：右下 6px 6px 14px 冷灰（#C8CDD3），左上 6px 6px 14px 纯白
- 24px 圆角，无可见边框
- 内含小图标与标签

六张卡片均使用 Apple 蓝（#007AFF）图标：
- 「研究规划」— 路线图图标
- 「语义检索」— 放大镜图标
- 「多 Agent」— 互联节点图标
- 「投稿管理」— 纸飞机图标
- 「知识图谱」— 网络图图标
- 「论文精读」— 翻开书本图标

主操作按钮使用蓝色渐变（145°，#1A8AFF → #0062CC），搭配蓝色发光拟态阴影（蓝色调阴影 + 浅蓝高光），形成标志性的发光按钮效果。

排版（左上区域）：
- 主标题：粗体中文无衬线字体（苹方 / SF Pro），「从论文堆里，拉出清晰研究路线」— 深藏青色（#1A2233）
- 副标题：较细字重，「面向科研学习与论文工作的桌面端 AI 研究助手」— 柔灰色（#6B7480）
- 品牌标识「小妍」配小头像图标置于角落

底部条栏：四个胶囊形平台按钮（「macOS」「Windows」「Linux」）带拟态按压效果（内凹阴影），底色略深（#E2E6EC）。

整体风格：拟态 UI 插画，清爽通透，Apple 级高级感，冷调蓝灰色系，柔蓬深度无硬边。专业但亲和。参考 Apple Human Interface、neumorphic.io 与现代 SaaS 产品落地页。

配色：冷蓝灰底（#F0F4F8 → #E8ECF0），深藏青文字（#1A2233），Apple 蓝主色（#007AFF），拟态阴影（#C8CDD3 / #FFFFFF）。

比例 16:9，高分辨率，无水印。
```
