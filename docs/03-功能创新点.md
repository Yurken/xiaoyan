# 小妍 — 功能创新点

## 一、协作式研究规划（Planner）

### 1.1 功能描述

用户输入研究主题后，系统启动 Topic Analyst / Paper Scout / Learning Path Designer 多 Agent 协作流程，输出：

- **学习路线与阶段拆分**：前置知识 → 核心阶段 → 进阶方向
- **关键文献线索**：每个阶段的代表论文与必读材料
- **可行切入点**：基于当前研究热点的具体入手建议
- **后续行动建议**：明确的下一步操作（检索、阅读、实验）

### 1.2 创新点

- **可视化协作流程**：展示各 Agent 的协作步骤与输出，不是黑盒生成
- **与综述联动**：规划结果自动作为综述模块的检索语义输入
- **方向资产化**：规划结果保存为"研究方向"对象，后续论文、笔记均可关联

---

## 二、协作式文献综述（Survey）

### 2.1 功能描述

围绕用户研究问题，启动 Intent Planner / Literature Retriever / Survey Writer 多 Agent 协作，输出：

- 结构化综述初稿（含章节划分）
- 候选论文列表与关键争议点
- 与研究方向目录联动沉淀

### 2.2 创新点

- **流式生成**：综述文本逐段流式输出，用户可边读边停
- **RAG + 知识库联动**：基于本地论文库生成综述，引用真实论文
- **写作素材沉淀**：综述草稿可作为投稿模块的写作素材

---

## 三、论文精读与图表关联分析

### 3.1 功能描述

PDF 导入后，系统自动：

1. **文本提取**：全文提取并分块向量化
2. **图表提取**：双路提取位图 + 视觉扫描矢量图/表格
3. **AI 精读**：从 Motivation、Method、Experiment、Limitation 等维度分析
4. **复现指南**：生成代码级复现步骤、环境配置、数据集链接

### 3.2 创新点

- **图表编号引用**：AI 分析时主动引用 "如图 3 所示"，图文并排展示
- **复现指南**：无开源实现时主动给出替代复现方案
- **自动重命名**：导入时按模板（作者-标题-年份）就地重命名原文件
- **关联研究方向**：论文可归属到具体研究方向，支持分组管理

---

## 四、知识图谱与笔记双视图

### 4.1 功能描述

知识模块提供两种互补视图：

- **知识图谱视图**：观点、证据与关联关系的图结构展示
- **笔记视图**：结构化知识条目，支持标签归档与语义搜索

### 4.2 创新点

- **图谱 → 笔记联动**：图谱中的节点可直接展开为笔记编辑
- **论文分析结论一键沉淀**：论文精读结果可快速转为知识节点
- **语义搜索**：基于 Embedding 的笔记语义检索，不只是关键词匹配
- **按方向聚焦**：在特定研究方向下过滤图谱，减少噪音

---

## 五、可观测的小妍协同台

### 5.1 功能描述

小妍协同台是多 Agent 协作入口，核心强调**"过程可见"**：

- **会话管理**：多会话并行，上下文绑定研究方向
- **任务总览**：Agent 状态图与运行轨迹面板
- **计划步骤**：每个任务的执行计划与当前进度
- **技能选择**：用户可主动选择特定技能（如"深度分析某篇论文"）

### 5.2 创新点

- **Mission Control 面板**：类似操作系统的任务管理器，实时展示各 Agent 状态
- **中间产物查看**：可查看每个 Agent 的原始输出，不只是一句总结
- **流式状态更新**：SSE 推送执行状态，前端无需轮询

---

## 六、投稿全生命周期管理

### 6.1 功能描述

覆盖从"写完论文"到"成功发表"的完整流程：

| 阶段 | 功能 |
|---|---|
| 选刊 | 刊会追踪、智能推荐 |
| 准备 | 提交清单、版本管理 |
| 审稿 | AI 模拟审稿、审稿意见归档 |
| 回复 | 审稿轮次跟踪、Cover Letter 辅助 |
| 追踪 | Kanban 看板、DDL 日历 |

### 6.2 创新点

- **版本快照**：每个版本的 PDF 和文本均可回溯对比
- **AI 模拟审稿**：投稿前预审，提前发现潜在问题
- **拒稿转投恢复**：拒稿后一键转投，保留历史进度
- **时间线视图**：投稿历程可视化回溯

---

## 七、实验记录与工作区

### 7.1 功能描述

- 实验记录创建与版本化编辑
- 配置、结果、备注结构化记录
- 图片/截图附件上传与预览
- 与投稿条目关联追踪

### 7.2 创新点

- **证据链管理**：实验结果与论文、投稿建立关联索引
- **图片证据**：支持截图上传，实验结果可视化归档
- **可追溯**：实验日志支持复盘与写作取证

---

## 八、实用工具集

### 8.1 功能清单

- **arXiv 智能检索**：关键词 + 时间窗口 + 三步级联筛选 + LLM 重排
- **期刊分区查询**：WoS / JCR / 中科院，支持 10+ 种等级动态过滤
- **CCF 等级查询**：679 个会议与期刊即查即得
- **学术翻译**：专业学术文本翻译
- **Markdown 格式化**：文稿整理
- **PPT 生成与预览**：一键生成 PPT，内置幻灯片预览，支持导出 .pptx
- **科研友链**：按分类整理的学术资源导航

### 8.2 创新点

- **检索范围自动填充**：选择研究领域 + 等级后自动填充 arXiv 分类和期刊范围
- **PPT 工作流内置**：从研究内容到演示文稿，无需切换工具
