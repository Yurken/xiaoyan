## 快速定位

- **如果你正在准备答辩 PPT**：先看 `05-答辩PPT大纲.md`，再按需查阅 01-04 提取素材
- **如果你需要回答技术问题**：重点看 `02-技术创新点.md` 和 `04-架构设计亮点.md`
- **如果你需要做项目介绍**：以 `01-项目概述与核心亮点.md` 为主
- **如果你正在规划后续迭代**：参考「后续规划文档」中的四份规划
